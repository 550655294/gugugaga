## 🧠 即梦 Seedance 2.0 核心技巧

### 五要素导演法（官方框架）

```
角色指定 → 谁在画面里？
场景设定 → 故事发生在哪？
动作/剧情 → 发生了什么事？拆成工序！
镜头语言 → 怎么拍？推拉摇移跟
氛围/声音 → 整体什么感觉？
```

### 六大原则

1. **素材分工** — 上传时说清每个素材是干什么的
2. **动作拆工序** — ❌"做饭" → ✅"切菜→下锅→翻炒→盛出"
3. **镜头语言** — "推近特写"比"放大"好10倍
4. **时间轴控节奏** — 长片按秒分段写指令
5. **参考+改造** — 用别人视频的运镜，替换自己的角色
6. **行为表达情绪** — AI不懂"伤心"，但懂"低头""擦眼泪"

### 崩了怎么办

1. 减少动作数量（每次1~2个动作）
2. 加"缓慢地""轻微地""小幅"
3. 先试文生视频（不上传参考图）
4. 缩短每段时长到3~4秒

## ✂️ 剪映后期流程

### 配音

| 方法 | 操作 |
|------|------|
| **剪映AI** | 文字朗读 → 奶萌女童音色 → 语速0.8倍 |
| **导入原版** | 爱给网下载的咕咕嘎嘎MP3导入 |
| **自录变声** | 手机录音 → 剪映变声 → "萝莉" |

### BGM
- 搜 `可爱日常` `日系小清新` `轻松治愈`
- 反转处突然静音0.5秒
- 音量 **-12dB**（不盖人声）

### 字幕 & 音效 & 导出

- 字体：站酷快乐体/喵呜体 + 白色描边3px + 弹入动画
- 音效：搜 `duang` `弹簧` `啪嗒啪嗒` `星星`
- 导出：1080P / 30fps / MP4 / H.264 / 9:16竖版
