## 提示词 1：咕咕嘎嘎做早餐（30秒）

```
杰作，超可爱的3D萌系动漫动画，温柔治愈风格。30秒竖屏。

【0-5秒 厨房全景→登高】
日式厨房，清晨六点半阳光从格子窗斜洒入，浅木色地板上形成方形光斑，光束中金色微尘缓慢翻滚。台面上木砧板、两颗鸡蛋在玻璃碗中泛温润光泽。企鹅妹妹系着粉色小围裙（背后蝴蝶结比肩膀还宽，绣着简笔画小鱼），踮脚来到灶台前——头顶刚与台面齐平。她踮起脚尖脑袋冒出三厘米、落下又沉回台面以下，上上下下弹了三次如打地鼠。默默拖来天蓝色小板凳爬上去——一条腿先跨上挂住蹬三下才翻上来。

【5-12秒 鸡蛋战役】
她翅膀尖拿起第一颗蛋在碗边轻敲——太轻纹丝不动。再敲——力度过大整个蛋在翅膀里炸开，蛋液从翅膀缝流下在晨光中如琥珀色宝石串滴落。第二颗蛋用喙辅助敲——蛋壳全掉进碗里。她歪头盯着碗中碎壳，小翅膀尖一片片往外捡——每捡一片凑眼前确认→放旁边形成小蛋壳堆。捡到第七片时蝴蝶结挂到蛋壳堆，转身时整碗蛋液被扫翻——蛋液划出慢镜头弧线精准流进灶台与墙壁夹缝，消失得干干净净只剩一道亮晶晶痕迹。她趴在灶台上看着消失的蛋液，眼中慢慢蓄起薄薄泪光——在睫毛上凝成小水珠，下唇不受控制地轻微颤抖。

【12-18秒 重启→成功】
她吸吸鼻子用翅膀背面擦眼（动作很大力把眼角擦红），深呼吸——身体随吸气鼓成气球→随呼气缩小。拿起最后一颗蛋双手捧如易碎宝物，用喙尖端在碗沿极轻碰一下——"咔"完美裂痕。双翅尖各捏蛋壳一边，闭一只眼对准碗中心缓缓掰开——蛋液完整滑入碗中，蛋黄圆润饱满。静一秒后她缓缓睁眼——眼中高光一颗颗重新亮起如夜空星辰。原地蹦起弹了三下，蝴蝶结上下飞舞，双手高举过头想做胜利姿势——手太短像头顶比了一个迷你V字。

【18-26秒 煎蛋艺术→享用】
开火——蓝色和橙色火苗从炉心升起，平底锅里倒一圈油（油瓶比她手还大，双手抱住往锅里灌差点趴进锅里）。她端碗如临大敌倾斜碗口，蛋液滑入热油——"滋啦"一声边缘泛起金黄蕾丝花边，蛋白从透明快速变白。翻面时锅铲是她三倍长——铲子插蛋下抬不起来→用力过猛蛋飞起空中翻一圈完美落回。她不敢置信看锅铲，然后又翻了七八次蛋越飞越高。蛋最终落盘：边缘微焦形成金色光环，溏心在晨光下微微颤动。她用番茄酱挤了微笑表情——弯弯眼加歪嘴。端着盘子爬上椅子（爬得费劲挂住蹬三下），餐巾塞进领口垂到地上。叉子插破溏心——金色蛋黄液如熔岩缓慢溢出铺满蛋白。送进嘴里——闭眼一秒猛然睁眼，瞳孔炸开无数小星星。身体不由自主左右摇摆，呆毛画圈，嘴里含糊不清发出甜腻的"gugugugugu~"。阳光从侧面勾勒她毛茸茸轮廓，蝴蝶结投下心形阴影。最后定格——闭眼微笑咀嚼，一边脸颊鼓包，嘴角沾一粒蛋黄屑，睫毛被阳光染成金色。

柔晨光左侧打，奶黄色调，微尘漂浮，景深虚化窗外 --ar 9:16 --motion 2 --stylize 200
```

## 提示词 2：咕咕嘎嘎下雨天（28秒）

```
杰作，最佳品质，3D萌系动画，雨后安静氛围。28秒竖屏。

【0-5秒 雨后巷弄】
灰蓝色调的日式小巷，雨刚停，薄白雾在石板路面高度飘浮。青石板到处是水洼——大的如镜倒映电线杆天空，小的闪着碎钻亮光。枫树叶挂满雨滴，每滴如微透镜折射周围景物，偶尔滴落激起同心圆涟漪。远景一只花猫在屋檐下舔爪子。左下角——明黄色小雨靴踏入画框，"啪嗒"响。镜头上摇：企鹅妹妹撑透明小雨伞（伞柄为小鸭头造型，透明PVC面可见雨滴滚落），穿着配套黄雨衣，帽上有企鹅耳朵尖角。她站水洼前低头看自己倒影——倒影也抬头看她，身后是被雨水洗透的湛蓝天空。

【5-12秒 踩水】
她抬脚在空中停顿一秒如瞄准——"啪"踩入水洼正中，水花不高但四面均匀溅开，每滴飞溅水珠在慢镜头中折射七色微光。她"咕咕！"惊喜短笑，开始水洼中反复跳——左右脚交替节奏越来越快，水花越溅越高。伞在跳跃中摇晃把雨滴抖落形成小型人工降雨。她开始旋转——伞面上的水珠被离心力甩出形成一圈圆形水帘。路过的蜗牛在石板缝中伸出触角，触角小球体映着她旋转的身影。笑声"gugugugugaga"越来越响在湿润空气中清脆传开。

【12-18秒 风吹伞翻→接雨】
一阵急风吹过——枫树枝叶剧烈摇晃积攒的雨滴哗啦啦同时落下。她伞被风从下掀起——"啪"透明伞面翻成倒扣的碗。真正的雨又飘起来，细密雨丝直直落在她头上、睫毛上、鼻尖上、喙上。她缩起脖子眯眼，呆毛被雨淋贴额头如海带。然后——突然仰头张嘴对天空接雨水。一滴精准落舌头，她如品红酒般含一下后眼睛睁大。开始更积极地仰头追雨：左移接一滴，右移三步接另一滴，摇摇摆摆追雨玩。花猫从屋檐出来坐她旁边，仰头看她疯狂追雨，耳朵抖了抖，打了一个哈欠。

【18-23秒 天晴蓝光】
雨渐小，云层裂开——第一道阳光如聚光灯射下，金色光束精准照在她和翻面伞上——翻面伞接满雨水变成透明"碗"。阳光穿过水——在伞下的她脸上投出波光粼粼的淡蓝色光斑。她低头看伞中水洼——水中除了她自己还倒映出雨后的蓝天白云和一道小彩虹（虹从伞边缘弯进水里）。一只蝴蝶飞过——她追逐蝴蝶跑出水洼，蝴蝶停在枫叶上。她靠近时蝴蝶翅膀开合一次，她下意识跟着眨了一次眼——同步。

【23-28秒 结尾】
彩虹完全显现横跨整条巷子上方。她站巷子正中仰头看彩虹——翠绿眼眸中倒映七彩光弧。花猫踱过来蹭她的小雨靴。她蹲下：把翻面伞平放在青石板上——伞中积的雨水变成一面圆形水镜，正对天空完整倒映整道彩虹。她指伞中彩虹给花猫看："gugu..."。花猫伸爪轻轻碰伞中水面——涟漪扩散模糊彩虹又缓缓恢复清晰。最后定格——远全景：小巷/彩虹/蹲在伞边的小企鹅和猫的背影/满地水洼里闪烁的碎光。

灰蓝转金色的自然光变化，湿漉漉的质感，水珠在睫毛上闪烁，雨后白噪音氛围 --ar 9:16 --motion 2 --stylize 200
```

## 提示词 3：咕咕嘎嘎睡午觉（25秒）

```
杰作，超可爱的3D萌系动画。25秒竖屏，极度舒适午睡氛围。

【0-6秒 午睡全景】
午后两点——白色纱帘被微风吹起轻柔飘动，阳光透过帘子在地板上投下流动的条纹光斑。一只企鹅妹妹趴在地毯正中央，身体蜷成一个完美的毛茸茸球——黑白配色像一颗巨型汤圆。镜头缓缓推近：她的身体随着呼吸规律地起伏——吸气时圆球胀大两圈，呼气时缩回，循环如潮汐。翅膀尖在睡梦中轻轻抽动——尖端三根羽毛依次弹起弹落，像在钢琴键上轻敲。嘴里断断续续发出细小的梦话"嘎……嘎……"，每一声"嘎"时喙轻微开合，嘴边枕头布面上已有一小滩口水画成乒乓球大小的圆圈。一颗靠垫被她当成抱枕夹在两条小短腿之间，靠垫上绣的胡萝卜图案被挤压成波浪形。

【6-13秒 阳光骚扰】
阳光光斑缓慢在地板上移动——爬过地毯边缘→越过她脚→爬上她后背→停在她脸上。强光直射眼皮——透过去隐约可见她眼球在眼皮下快速转动。她皱了皱小鼻子——鼻头抽动三次（带动呆毛跟着同步震颤），然后下意识伸长一侧翅膀当遮阳棚盖住脸。但翅膀羽毛间有缝隙——阳光变成条纹从缝隙漏下来像斑马纹。她不满地哼"嗯——"，翅膀调整角度但总有光漏进去。于是她翻身——身体从左侧滚向右侧——结果连续翻了两圈滚出了地毯范围滚到光秃秃的木地板上。接触到冰凉地板一瞬间她身体明显打了个寒颤——全身羽毛同时炸开一瞬然后收拢。

【13-20秒 迷糊寻毯】
她迷迷糊糊睁开一只眼睛——眼白里充满血丝，瞳孔还没对焦，视线里的世界是模糊的一团。伸出一只翅膀摸了摸身下——光秃秃冰凉。她闭回眼睛，身体开始像虫子一样蠕动——匍匐前进往地毯方向挪。每挪一次抬头闭眼用翅膀探路（翅膀尖在空气中左扫右扫如探测器），探测到地毯边缘时呆毛先弹起——然后整个人往侧边一滚精准落回地毯上面。找到正确位置后她用小翅膀拍拍地毯确认——是自己之前趴过的位置有条形光斑温度刚好——然后满足地叹一口气把脸埋进靠垫里，靠垫上胡萝卜图案弹回原位。三秒后呼吸恢复均匀——一秒就睡回去了。

【20-25秒 最后三秒】
远处隐约传来夏天的蝉鸣——声音随着窗帘的飘动飘进来。画面安静地停在她身上——地毯/光斑/蝉鸣/均匀呼吸/随风轻飘的白纱帘/她嘴角挂着一丝极小弧度仿佛梦见了好事。最后三秒一只蝴蝶落在窗帘上——翅膀的开合节奏和她呼吸频率恰好一致。画面缓缓淡出。

暖黄午后日光，慵懒的蝉鸣白噪音，极度舒适催眠氛围 --ar 9:16 --motion 2 --stylize 200
```

## 提示词 4：咕咕嘎嘎拆快递（28秒）

```
杰作，3D萌系动画。28秒竖屏，温馨搞笑拆快递日常。

【0-6秒 巨箱降临】
明亮客厅，阳光充满整个房间。画面中央——一个巨大的瓦楞纸快递箱横在地板上，尺寸惊人——企鹅妹妹站旁边只到箱子三分之一高。她绕箱走一圈——小翅膀背在身后像巡查员，从左边走到右边花了好几步，走完停下用翅膀尖敲敲纸箱——声音很闷表示里面东西很多。她仰头看箱顶——脖子仰到极限身体差点后翻，然后纵身一跳——贴箱壁滑下来。单面胶带封口在箱顶——对她来说是遥不可及的登顶挑战。她眯起眼露出不服气表情，呆毛一翘开始搬救兵：从墙角拖来小板凳（回头看一眼箱子确认高度不够），又摞上一本厚字典（再回头——还差一截），最后顶上一个毛绒靠垫（再回头看——刚刚好）。三个物件叠成的"塔"摇摇晃晃，最上面的靠垫已经歪斜。

【6-13秒 塔塌→换策略】
她深吸一口气开始攀爬。第一步踩小板凳——稳。第二步踩字典——字典封面太滑她差点劈叉，拼命用翅膀保持平衡。第三步踩靠垫——靠垫是软的一只脚陷进去整个人歪了，"塔"开始剧烈摇晃，她在顶上像踩冲浪板一样保持平衡——然后"哗啦"一声整座塔塌了，她和靠垫一起摔进旁边一堆泡沫包装纸里。泡沫纸炸开飘满画面——在这片白色海洋中两只黄色小脚从泡沫纸堆里伸出来在空中无助地蹬踹，伴随闷闷的"嘎……嘎……"。

【13-20秒 拆箱方法】
她从泡沫纸堆挣扎出来——头上身上挂满泡沫碎屑，头发里还插了好几颗泡沫花生。甩头——泡沫碎屑四处飞散。决定换策略：不再攀爬，改用喙——黄色大喙对准纸箱侧面胶带接缝处一口咬下，侧头撕拉——胶带沿直线被撕开。她张嘴松开胶带——胶带黏在喙上她甩了好几下才甩掉。再用喙撬开纸箱盖一角——盖子翻开露出里面。她踮脚趴在箱子边缘往里看——身体前倾角度越来越大最后双脚离地整个人挂在箱缘悬空，像一只趴在悬崖边的企鹅。

【20-28秒 收获与满足】
她直接翻进箱子里——箱内传来一阵熙熙攘攘的翻找声，偶尔冒出一个小气泡包装袋被捏破"啵"声。然后她满头大汗地从箱子里爬出来——怀里紧紧抱着一个比她身体还小的迷你小鱼干抱枕。抱枕是浅橘色鱼干形状，上面绣了眼睛和腮线，尾巴是流苏。她抱紧抱枕把脸深深埋进去——幸福地眯起眼睛，左右摇晃身体，嘴里发出"gugugugugu~"像在哼摇篮曲。阳光照在她和抱枕上——满地的泡沫碎屑在光中闪闪发光如星星散落一地。最后镜头拉远——她抱着小鱼干抱枕躺在地板上睡着了，旁边是半开的大纸箱，泡沫碎屑像雪一样铺满地板，其中一颗泡沫花生还粘在她的呆毛尖上。

明亮客厅日光照，泡沫碎屑在光束中闪烁如星 --ar 9:16 --motion 2 --stylize 200

## 提示词 5：咕咕嘎嘎浇花（26秒）

```
杰作，超可爱的3D萌系动画。26秒竖屏，黄昏阳台治愈时刻。

【0-6秒 阳台全景→浇花日常】
夕阳时分的公寓阳台。木制花架上摆满各种多肉植物和小盆栽——彩色陶盆排成弧形，每盆泥土上插着迷你名牌（歪歪扭扭写着"小绿"、"胖胖"、"刺刺"等）。远处城市天际线在橙色晚霞中变成剪影。企鹅妹妹端着一个比她头大两倍的蓝色洒水壶从室内推开门出来——水壶太重她走一步晃三步。她开始按顺序浇花：第一盆"小绿"（一株圆滚滚的多肉，叶片像绿色水滴）——浇水时轻声说"gugu~"并在水壶嘴上控制水流让它画出一个心形水圈；第二盆"胖胖"（仙人球）——浇完后检查泥土湿度用翅膀尖戳了戳；第三盆是三株一排的小雏菊——她给每一株单独问候"gugu"、"gaga"、"gugu"。

【6-13秒 仙人掌对话】
来到最后一盆——比所有都大一号的仙人掌，陶盆上名牌写着"刺刺"，旁边加了括号小字"（脾气不好）"。她犹豫了——水壶嘴伸过去又缩回来，重复三次。最后只挤了一滴——精准滴在花盆边缘离仙人掌本体最远的位置。然后她凑近仙人掌（距离保持安全，鼻子和刺之间隔了至少三厘米），压低声音悄悄说："你少喝点，你扎人。"说完还往后退了一步。此刻——仙人掌顶端那个含苞已久的小花苞突然微微颤动，然后缓缓绽放——花瓣一片片展开，颜色从淡粉渐变到玫红最终在花心处凝成深红色。花绽放全程约三秒，每一片花瓣展开时都有一小圈肉眼可见的光粉扩散。她瞬间双手捂住嘴巴——眼睛瞪大到极限，瞳孔中倒映着正在绽放的花朵，高光如烟火般依次炸开。

【13-20秒 奔走相告】
