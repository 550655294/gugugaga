# 鬼畜魔性风格提示词

> 风格来源：涂山白白喵 / B站热门鬼畜区
> 核心：短小高能、魔性循环、荒诞反差
> 时长：20-30秒（适配抖音鬼畜节奏）

---

## 风格特征

- ⚡ 节奏：快切快推，镜头运动剧烈
- 🎵 音频：gugugaga原声（婴儿呓语调）+ 节奏卡点
- 🎭 反差：可爱外表 + 荒诞行为
- 🌀 结构：重复循环 + 一次反转爆点

## 提示词 1：咕咕嘎嘎健身房

```
杰作，最佳品质，3D萌系Q版动画，流畅且富有弹性卡通物理。30秒竖屏短片。

【0-3秒 开场 — 自信登场】
画面以白色干净直播间风格为背景，地面是光滑的浅灰色健身垫，右侧一台黑色跑步机静止待命，顶部一盏环形补光灯投下柔和白光。企鹅妹妹从画面左侧昂首挺胸迈进来——黑色毛茸茸身体圆滚滚，纯白肚子微微前挺，鹅黄小嘴扬起自信弧度，翠绿色超大圆眼多层高光闪闪，黑色齐刘海波波头随步伐一弹一弹，头顶一根呆毛高高竖起如天线。她换上了一件荧光粉色运动背心（明显大了两号，肩带不停往下滑），脖子上还挂了一条白毛巾，翅膀端各套了一个迷你举重手套（手指位置空荡荡）。镜头从低角度仰拍，让她看起来格外高大，背景加上金色光芒滤镜，带轻微慢动作效果——每一步落地的瞬间健身垫微微下陷回弹。她站上跑步机，双手叉腰（手太短只能撑在肚子两侧），对着镜头竖起一根羽毛："guga！" 一滴自信的汗珠从额头闪亮滑落。

【3-8秒 跑步机暴走 — 灾难降临】
特写镜头：她的翅膀尖按下启动键——屏幕数字从1跳到10只用了0.3秒。跑步机发出轰鸣声，传送带飞转。中景：她的小短腿开始疯狂交替——先是正常跑，然后速度快到双腿变成模糊的螺旋桨状残影，整个身体因为跟不上速度开始剧烈抖动，毛茸茸的身体像果冻一样波浪形震动。面部推近特写：表情从自信微笑→嘴角僵硬→眼睛瞪大到极限（翠绿瞳孔缩小成针尖），嘴巴张成完美的O形。呆毛被风压吹得笔直向后贴平。然后啪——她的一只脚踩空，整个身体被传送带向后弹飞。大全景慢动作：她在空中翻了整整两圈，翅膀像风车一样旋转，毛巾和手套在空中脱手飘浮。背景闪过夸张的速度线和放射状冲击线。最后"砰"一声——她像一张饼一样拍在后面的白墙上，身体完全摊平，四肢和喙在墙面上形成一个完美的企鹅形凹痕。墙灰簌簌落下。

【8-12秒 变成企鹅饼 — 滑落】
墙面特写：她贴在墙上像一张海报，眼睛变成了两个巨大的旋转XX螺旋圈（卡通晕眩符号）。一只手套从空中缓缓飘落，精准地扣在她头上。白毛巾落在跑步机传送带上被卷入机器，发出吱吱嘎嘎的声音然后被吐出来变成一条扭曲的麻花。她开始缓慢地从墙上滑下来——先是翅膀尖离开墙面，然后是喙（发出类似橡胶剥离的"啵"声），接着是整个身体像融化的冰淇淋一样沿着墙面往下淌。滑到地面时她瘫成一滩——完全扁平状，像一滩黑白色的液体，只有两只XX眼还在一闪一闪。跑步机还在空转，屏幕上显示"卡路里消耗：0"。

【12-18秒 恢复与挣扎】
一滩企鹅液体开始缓慢重组——先是身体从二维弹回三维（伴随"啵"的卡通音效），然后是四肢恢复原状。她摇摇晃晃站起来，头发乱七八糟像被台风吹过，脸上还有墙灰印。运动背心已经歪到一边挂在一只翅膀上。她看了看跑步机（镜头给跑步机一个缓慢推进的恐怖片低角度）——跑步机还在转动，屏幕上的数字闪烁变成了恶魔般的红色。她后退一步，小腿开始发抖。然后深吸一口气，重新爬上去——这一次她抓着两侧扶手，只用脚尖轻轻试探传送带。刚迈出第一步，跑步机突然射出火花，变得更快。她再次被弹飞到墙上——这次在墙上留下了一个更深的凹痕，凹痕旁边已经有好几个企鹅形状的旧印子。

【18-24秒 最终暴走与结尾】
决定性时刻：她从墙上挣脱，眼睛里燃起火焰（真的有两团小火苗在瞳孔里燃烧）。她走到跑步机侧面，举起一只翅膀——翅膀尖伸长变成一柄巨大的卡通木槌（锤头上画着禁止符号）。高高举起——停顿一秒让木槌蓄力发光——然后猛砸下去。跑步机冒出一团黑烟，屏幕碎裂。黑烟散去后——跑步机变成了一堆零件，传送带像舌头一样耷拉在外面。她拍拍翅膀上的灰，扛起木槌架在肩上，转身对镜头露出胜利微笑。此时墙上出现一行大字字幕"DAY 1"，然后画面快闪一连串画面——墙上凹痕越来越多，跑步机零件堆成小山，"DAY 47"出现在屏幕中央——她瘫在零件堆顶上，面前竖着一面小白旗，上面画了一个鱼干图案。最后画面定格在她的特写——脸上脏兮兮但眼睛弯成月牙，歪头："gu...ga？"

## 提示词 2：咕咕嘎嘎抢红包

```
杰作，最佳品质，3D萌系动画。25秒竖屏短片。

【0-3秒 设置】
暖黄色客厅，企鹅妹妹趴在毛绒地毯上，面前是放在手机支架上的巨大手机（屏幕比例夸张放大）。她双翅前伸呈蓄势待发状，眼睛紧锁屏幕，瞳孔里映射出微信红包界面的倒影——一个红色气泡在屏幕中央闪烁倒计时"3...2...1..."。她的呆毛如雷达同步倒数旋转。周围空气凝固，窗外的风都停了，窗帘定格。随着倒计时到0——她所有羽毛同时炸开一瞬。

【3-7秒 第一轮】
红包弹出——她翅膀以肉眼不可见的速度爆发点击——画面用慢动作展现：翅膀尖的羽毛因摩擦发热冒出细烟，屏幕玻璃因高频点击出现微裂纹。但屏幕弹出冰冷的灰色提示："红包已抢完"。她愣住了——身体保持点击姿势彻底石化，只有眼睛眨了一下。背景音效骤然切断，一片死寂。镜头缓慢推进她的侧脸——眼睛里的高光一颗接一颗熄灭（一共四颗，从下往上依次熄灭），呆毛像被戳破的气球一节节软塌下来。她的嘴巴开始缓慢鼓起——两腮像充气一样膨胀，从淡粉变成气鼓鼓的通红。一片枯叶从画面中凄凉飘过。

【7-12秒 暴走爆发】
她猛地站起来——动作快到身体在原地留下一个残影。然后深吸一口气（吸气量巨大到整个房间的窗帘都被吸得朝她飘过来，茶几上的纸巾盒朝她滑动了几厘米），张嘴对着手机屏幕爆发："GU——GU——GA——GA！！！" 每一个音节从她嘴里喷出时都变成可见的冲击波——第一声"GU"让手机屏幕闪了一下，第二声"GU"让手机弹了起来，"GA"让手机在空中翻了个跟斗，"GA"让手机在半空中停住。然后奇迹发生——喊声每击中屏幕一次，上面就弹出一个红包——不是一个一个弹，而是像井喷一样几百个红包同时涌出，像红色的潮水一样从屏幕里漫出来。红包堆积成山，淹没了她的脚踝→膝盖→肚子→脖子。她惊慌失措地挣扎但被红包的洪流越推越高，最后整个人被顶到天花板上，四肢张开贴住天花板像一只被钉在墙上的飞镖。

【12-18秒 红包废墟】
画面变为一片红色的红包海洋——客厅地板完全被覆盖，只露出沙发靠背和台灯的灯罩。红包堆中伸出一只黄色小脚，在空中无助地蹬了几下。然后她用喙从红包堆里凿出一个洞爬出来——头发里插满了各种红包皮，呆毛上挂着一个小号的"0.01元"标签。她环顾四周的红包废墟，表情呆滞。颤抖着伸出翅膀点开了其中一个——屏幕上跳出"0.01元"。再点一个——"0.01元"。她开始疯狂地拆红包——拆一个扔一个，像小鸡啄米一样机械重复——每一个都是"0.01元""0.01元""0.01元"，0.01的硬币从屏幕里哗哗掉出来在她身边堆成了一个小钢镚堆。一个巨大的金色"总计：0.47元"的牌子从天上掉下来咣当砸在她面前。

【18-25秒 结局】
她跪坐在满地红包和钢镚中——整个人褪去了所有颜色变成了灰白色（只有眼睛保持翠绿）。突然手机又响了——一个新的红包弹出，金色的，比之前所有的都大，带翅膀会飞。她的颜色瞬间恢复，眼睛变成超大的星星——扑过去。点开——屏幕显示："手慢了，红包派完了"。她变成石头——从脚开始向上石化，到脖子停住。然后石头碎裂——她整个人散成一地红包。最后一张红包从空中飘落在废墟顶端——上面是她的Q版头像，配字："抢了个寂寞"。

## 提示词 3：咕咕嘎嘎面试

```
杰作，3D Q版动画，夸张的卡通形变动画风格。28秒竖屏短片。

【0-4秒 开场 — 办公室全景】
暖色调的现代简约办公室，落地窗透进黄昏的金色光线，空气中微尘缓缓飘浮。一张巨大的黑色办公桌占据画面中央，桌面干净到反光——只放了一杯冒着热气的咖啡和一份空白简历。镜头从桌面上摇至面试官位置——画面中始终看不到面试官的全貌，只有他巨大的影子投射在背后的墙上，再加上一双交叉放在桌上的手（手指修长，中指戴着一枚银色戒指）。画面右侧的玻璃门被轻轻推开——企鹅妹妹从门缝里探出半个头，翠绿大眼在门框阴影里闪闪发光。她穿着一件明显不合身的黑色小西装——袖子长得完全拖在地上像两条黑色的尾巴，领带打得比她的脸还大一倍（领结像一个大蝴蝶结横在胸前），里面白衬衫的袖子挽了八层才露出手。她小心翼翼走进来，每一步都踩在自己的袖子上，走一步绊一下，走三步差点脸朝地摔倒。最后终于摇晃着站在办公桌对面——坐下时因为腿太短整个人陷进椅子里只露出眼睛和呆毛。

【4-8秒 面试问题一 — "期望薪资"】
面试官的声音（低沉磁性男声）从画面外传来："你的期望薪资是多少？" 妹妹双手放在膝盖上（袖子太长手根本看不见），坐得笔直——翠绿大眼认真注视前方，呆毛竖得笔直像天线。她思考片刻，然后郑重其事地竖起三根羽毛（翅膀尖的三根羽毛像开花一样弹开）。特写：三根羽毛在灯光下闪闪发光，每一根尖端都挂着一颗小水珠。然后画面切到她的幻想气泡——三只被煎得两面金黄的香喷喷小鱼干在烤架上滋滋作响，冒着热气，一滴柠檬汁从上方滴落。她咽了一下口水，喉头明显地滚动了一下，认真点头："三条小鱼干。"面试官的影子在墙上顿了一下——咖啡杯在桌上发出了轻微的陶瓷碰撞声。

【8-12秒 面试问题二 — "加班"】
面试官："你能接受加班吗？" 她立刻点头——点得像啄木鸟一样飞快。但镜头捕捉到一个关键细节：她的翅膀在桌面以下偷偷动——镜头缓慢下摇钻到桌子底下。桌下的秘密世界：她的翅膀端着一支钢笔在一块小木板上疾书——上面画了一张夸张的逃跑路线图：一个企鹅形状的小人→钻过通风管道→顺着拖把杆滑下→跳窗→乘着纸飞机飞越城市→最终降落在沙滩上一把遮阳伞下，旁边画了一杯插着小伞的椰子汁。逃跑路线图右下角还标注了各段的预计耗时。镜头回到桌上——她脸上挂着标准职业微笑并对面试官露出八颗牙，但眼角在不自然地轻微抽搐。

【12-20秒 面试问题三 — "特长"爆发】
面试官："你最擅长什么？" 空气凝固三秒。她的微笑慢慢变僵然后消失，眼中开始旋转加载中的小圈圈（三个小白点依次闪烁）。然后她的眼珠从绿色瞬间变成金色——"叮"一声。她缓缓站起来（椅子因为释放的压力向后滑走撞到墙上）。开始：先是一个慢动作的深呼吸——吸气时整个身体像气球一样膨胀到两倍大，西装纽扣被崩飞一颗（嗖一声弹到镜头前）。然后爆发——她开始疯狂跳舞，全身每个关节都在独立运动：脖子像弹簧一样伸缩、身体像果冻一样波浪震动、翅膀交替高速旋转如直升机螺旋桨、两只脚交替原地踏步频率快到腿变成模糊的灰色阴影。她张嘴开始高速念"gugugagagugugagagugugagagugugaga"，音节从她嘴里以可见的音波形式弹射出来——每个音节变成实体字母在空中飘浮然后撞向四周。音波撞击文件柜——所有抽屉依次弹出；撞击白板——白板上的磁贴全部震落；撞击面试官的咖啡——咖啡表面泛起同心圆波纹最后溢出来。整个房间都在震动——天花板灯管忽明忽暗，百叶窗的叶片哗哗作响，墙上的画框歪了。面试官的影子在墙上剧烈颤抖，双手抓紧了扶手。

【20-28秒 尾声】
她完成最后的定格pose——单脚站立，另一条腿向后抬高，双翅展开呈V字，头高高昂起，呆毛弯成一个完美的钩。这个姿势保持了0.5秒后——身体肉眼可见地开始摇晃，然后像被推倒的积木一样直挺挺向后倒下。倒地后躺在办公室地板上大口喘气——嘴里还飘出最后一缕音节形成的白烟，白烟在空中拼成一个歪歪扭扭的"gaga"。办公室已是一片狼藉：文件散落一地像雪片，咖啡杯在桌边危险地旋转最后终于停稳，面试官的领带不知何时挂在了吊灯上。安静了整整两秒。然后面试官的双手从桌下缓缓举起一块打分牌——上面用马克笔写着"明天上班"，旁边还画了一个笑脸太阳。妹妹躺在地上一动不动，只有一只翅膀缓缓举起竖了一个大拇指，然后啪嗒落下。

## 提示词 4：咕咕嘎嘎照镜子

```
杰作，3D萌系动画，带轻微惊悚元素的喜剧风格。22秒竖屏短片。

【0-4秒 自恋日常】
