## 提示词 1：咕咕嘎嘎减肥记（喜剧反转·30秒）

```
杰作，最佳品质，3D萌系动画。30秒竖屏。

【前15秒 铺垫段 — 雄心壮志减肥】
厨房暖黄灯光，企鹅妹妹叉腰站在一台老式指针体重秤前——秤盘上有一层灰说明很久没用。她深呼吸，闭上左眼，小心翼翼先把右脚踩上去——指针微动一下——左脚也踩上去——指针疯狂转了两圈半后停在了一个数字上。画面切为她的主观视角：体重秤数字被加了超级厚的马赛克完全看不见，但旁边贴了一张便利贴写着"看了会伤心（上次看完哭了三天）"。她从背后掏出一张巨大手绘"减肥计划表"——表格画满红叉和哭泣企鹅emoji，标题加大加粗"第46次减肥"旁边画了一根小鱼干加禁止符号。她把计划表郑重地贴在冰箱门上——然后用一根短绳子把冰箱门绑了个死结。转身面对镜头握紧双翅："gugugaga！"——斗志燃烧表情，翠绿眼中冒出两团小火焰。蒙太奇：她在地板上做仰卧起坐（起一个倒三个气喘如牛，最后一动不动躺着看着天花板思考人生）→面对一盘绿油油沙拉发呆了半分钟后用叉子把每片菜叶都戳了洞但不吃→在跑步机上扶着扶手被传送带拖得脚离地面形成水平滑行→跳健身操时左右脚完全不在拍子上像在跳另一种舞。

【后15秒 反转段 — 深夜偷吃翻车】
时钟指针快进到凌晨十二点整——秒针滴答到12时画面全黑只剩时钟特写。然后一片漆黑中——冰箱门悄悄打开了一条缝。冰箱内部暖黄灯光透出如天堂圣光，照亮了她从门缝探出的半张脸——眼睛在暗处发出绿光，嘴角挂着邪恶微笑，呆毛弯曲成恶魔角形状。她先左右飞速探头确认安全——动作像一只偷坚果的松鼠，然后踮着脚尖（蹼足走路居然能做到无声）溜到冰箱前。用喙咬断之前绑的死结（绳子应声断成两截），冰箱门大开——内部全景：成排的小鱼干、三盒布丁、一整块奶油蛋糕、大桶冰淇淋、两罐巧克力酱——暖黄灯光把食物照得无比诱人，有柔和的辉光滤镜。开始暴风吸入——左手小鱼干右手蛋糕同时往嘴里塞，两腮鼓起如花栗鼠，嘴巴周围沾满奶油和饼干渣。突然——"啪"一声整个厨房灯光大亮——是那种审讯室级别的惨白顶灯全开效果。她僵住，嘴里还塞着没咽下去的食物。镜头缓慢摇到她身后——墙上原来密密麻麻贴满了微型摄像头。不是一个两个——整个后墙像监控中心一样，至少有十几个摄像头从不同角度对着冰箱。屏幕右下角浮现白色小字，像摄像机录像信息："摄像头37号 · 第47天 · 凌晨00:03:42"。她缓缓转身面对摄像头阵——满嘴塞满食物像塞满储存粮食的仓鼠，对着最近的一个镜头用含糊不清的声音说："g...guga？（能...删...删掉吗？）"嘴唇还在动但无声，一根小鱼干的尾巴从嘴角探出来晃了晃。画面定格在她的窘迫全脸——眼睛里旋转着尴尬的圆圈，随后淡出。
```

## 提示词 2：咕咕嘎嘎的生日（温馨反转·28秒）

```
杰作，3D萌系动画。28秒竖屏。

【前14秒 铺垫 — 孤独的生日】
傍晚的昏暗客厅——窗帘全部拉上只有一丝缝隙透进橘色路灯。企鹅妹妹独自坐在茶几前，面前一块巴掌大的小蛋糕——蛋糕上歪歪扭扭地插着一根数字蜡烛"1"。她环顾四周——沙发空荡/饭桌空荡/玄关空荡/没有气球没有彩带没有任何装饰。她低下头——头上戴的纸质生日王冠上的"生日快乐"小旗子也跟着垂下来，王冠边缘沾了一点她额头的汗渍。拿起手机点开消息界面——联系人列表一片灰色/0条未读。她盯着屏幕看了很久——屏幕暗了再点亮/暗了再点亮/第三次没再点。把手机轻轻翻面扣在桌上——手机壳背面是一张贴纸"今天是我的生日！"无人看见。自己从椅子后面摸出一面小镜子，对着镜子给自己戴上寿星帽（纸王冠本来就戴着，她又拿了一个叠在上面变成一个两层王冠）。点燃蜡烛——火柴划了三次才着，烛火在她翠绿瞳孔中轻轻摇曳映出两朵小火苗。她对着烛火小声说："gugu..."——声音在空荡客厅里形成微小回音。

【后14秒 反转段 — 惊喜变惊吓】
她闭上眼深吸一口气准备吹蜡烛——腮帮子鼓到极限——门"砰"一声被撞开。画面切到门口——一群企鹅蜂拥而入：领头的一只举着比她人还大三倍的彩虹气球束（气球轻飘飘顶着天花板）→第二只端着一个七层高的奢华蛋糕塔（奶油裱花极尽华丽/每层插不同颜色蜡烛/顶上站着一个糖艺企鹅）→第三只头上顶着一只用几百条小鱼干拼成的巨幅"生日快乐"字（鱼干排列整齐尾巴全部朝一个方向/中间还用心形罐头点缀）→第四只拿着彩带喷射器对准她就是一顿狂轰滥炸彩带从四面八方飞来。她的表情——先是惊恐瞪眼→然后眼眶开始泛红→嘴唇颤抖——所有人以为她要感动哭了——结果她喊的是"蛋——糕——！！" 慢动作展现：冲在最前面的胖企鹅因为地滑刹不住脚，连人带脚踩进了蛋糕塔——七层蛋糕像多米诺骨牌一样从下往上依次倒塌，奶油在空中画出螺旋，最后整个蛋糕塔扣在了她的头上。画面巨响后静止——她被奶油完全覆盖整个人变成一只白色的雪人，只有两只翠绿色的大眼睛在厚奶油中眨巴——睫毛凝着奶油霜。其他企鹅们集体石化——第三只的头上的鱼干"生日快乐"歪了一个字变成"生日不快"——角落里那只举气球的悄悄松手气球飞走天花板，然后默默退出房间把门轻轻带上。她伸舌头舔了一圈嘴周围的奶油——吧唧嘴："guga（甜的）。"随后一根蜡烛从奶油中重新燃起——火光照亮她裹满奶油的脸。

暗调独处转彩色派对爆炸转奶油混沌，冷暖色调对比 --ar 9:16 --motion 3 --stylize 200
```

## 提示词 3：咕咕嘎嘎拯救世界（英雄反转·28秒）

```
杰作，3D萌系CGI动画。28秒竖屏。

【前14秒 铺垫 — 末日降临】
黄昏的城市全景——高楼大厦远眺/平静的日常。突然一阵尖锐的防空警报响彻全城。高架桥上所有汽车急刹——司机们纷纷探头看天空。镜头沿着路人的视线缓缓上摇——突破云层后画面中一颗巨大的陨石正破空而下，拖着数公里长的橙红色火焰尾迹，陨石表面燃烧着岩浆般的裂纹发出地狱般暗红光。城市陷入恐慌——人群四散奔逃，街头屏幕全部切换为红色紧急警报。新闻直升机在空中颤抖盘旋，记者的声音从画面外传来（颤抖）"距离撞击还有三十秒……二九……二八……"。画面切到航拍——陨石阴影已经开始覆盖半个城市，地面因震动出现细碎裂纹。

画面切到企鹅妹妹的公寓——客厅茶几上摆着吃了半包的薯片和遥控器。她陷在沙发——电视正直播这场末日画面。她嘴里咬着一根小鱼干——咀嚼的节奏没有任何变化，和末日画面形成了诡异的平静反差。她慢条斯理地嚼完最后一口——舔干净每个翅膀尖（用了五秒），然后从沙发上滑下来（因为腿太短脚够不到地板看起来像滑下来）。走到鞋柜前——从里面找出一双她平时舍不得穿的闪光运动鞋——认真地把鞋带系成蝴蝶结——打了一个死结又解了半天。

【后14秒 反转段 — 史上最潦草的拯救】
她走到阳台上——夕阳和陨石火光双重打在她脸上形成半边暖半边红的诡异光效。她四处看了看——从晾衣架上取下一个最普通的彩色塑料晾衣夹（黄色的那种/夹子表面还有长期日晒产生的褪色纹路，之前是用来夹被单的）。她把夹子在翅膀间抛了两下试手感，然后深吸一口气摆出超级英雄的标准起手式——一手后拉一手指天/呆毛竖起/运动鞋闪光灯突然亮起。对着天空——把晾衣夹子投掷出去。夹子以不可思议的速度和力量冲向天空——突破音障产生一个小型白色锥形云——飞行轨迹是一条笔直的金线。

远景镜头：巨大燃烧的陨石（占据了画面百分之九十）vs 一个米粒大小的黄色塑料夹子（在画面中几乎看不见）。特写给到陨石表面——不知何时陨石表面长出了一个鼻子形状的凸起（有清晰的鼻孔）。夹子精准地——咔嚓——夹在了那个鼻子上。陨石发出一声惊天动地的"嗷————呜！！！"（痛苦嚎叫）。然后陨石在空中硬生生拐了一个90度直角——从城市上空擦着最高的摩天大楼天线向侧面飞过，消失在地平线尽头的云层里。只留下一条弯折的烟痕和空气中飘散的夹子塑料烧焦味。

她拍拍翅膀上的灰——从阳台走回房间，弯腰捡起遥控器关掉电视。看了一眼电视黑屏中的自己——头发有点乱但神情淡然。打开薯片袋子——发现吃完了——面无表情捏扁袋子放茶几，转身去厨房打开零食柜重新拿出一包。最后画面——她躺在沙发上咬着小鱼干，电视正在回放她夹飞陨石的新闻画面。主持人在电视里激动地喊"这是我们唯一的画面！"传回来的视频画面因为没人拍到具体经过只有一团模糊的光——主持人严肃地说"怀疑是外星科技干预"。她看着新闻面无表情地嚼了几秒，然后——举起遥控器换台变成一档烘焙节目。画面定格在电视上的草莓蛋糕——她咽了一下口水。

灾难大片级转无厘头喜剧，好莱坞严肃叙事 → 日式日常跳跃感 --ar 9:16 --motion 4 --stylize 200
```

## 提示词 4：咕咕嘎嘎表情包（次元壁破碎反转·26秒）

```
杰作，超可爱的3D萌系动画。26秒竖屏。

【前13秒 奇幻世界铺垫】
粉紫色调的异次元泡泡空间——天是渐变粉，地上是半透明蓝色镜面反射着天空。空间中到处飘浮着大大小小的半透明泡泡——每个泡泡里都活着一幅她的小小表情：开心（眼睛弯成月牙）、超级生气（圆鼓鼓脸像蒸包，头上冒出蒸汽）、困惑（小问号在头顶旋转还一个变三个）、打哈欠（张到最大的嘴能看到喉咙里的小舌头）、花痴（眼中两个巨大的心形取代瞳孔旋转）。她轻盈地在这个世界跳跃——每踩一下地面就泛起涟漪并有音符飘出。她跳起戳破一个"生气脸"泡泡——泡泡破裂时啪地散成七彩小光点和音符，表情缩成一个发光小图标飘进她的喙里。她戳了一个又一个——每戳破一个泡泡就收集一个表情——喙的边缘开始发出霓虹光。收集完毕后她站在地面镜面上低头看——水中倒影和真人开始隔空互动：她wink→倒影做飞吻/她做鬼脸→倒影扮酷/两人同时比爱心→爱心从水面浮出真的变成一个粉红实体桃心悬浮在空中被她接住。她抱着这颗巨大的爱心泡泡玩得不亦乐乎。

【后13秒 反转次元壁打破】
突然——画面剧烈震动像发生空间扭曲。泡泡世界的色彩从梦幻渐变被一股力量拉成线条状——所有泡泡同时破裂化为流水——整个世界像被卷进抽水马桶一样缩小缩小再缩小。最后画面从缤纷的泡泡世界缩小成一个普通智能手机的屏幕。一只人类的手拿着这部手机——拇指在屏幕上滑动。镜头拉远看清：手机屏幕上是微信表情商店页面，正显示"咕咕嘎嘎表情包第3弹——售价1元"的购买界面。拇指点击"下载"。画面瞬间切回泡泡世界——企鹅妹妹突然感觉到一股无形的力量支配了她的身体：她开始不受控制地做出各种表情——一个接一个像在拍大头贴。她试图反抗——"我又不是表情包！我要自由！！"——但身体被意志之外的力强制摆出剪刀手（wink表情）、嘟嘴（撒娇表情）、双手捧脸（期待表情）。最后她认命了——对着虚空中不知在哪的镜头摆出一个最完美的wink——单眼闭闭的那只眼角带一颗动漫式的闪亮星星，脸颊红晕恰到好处，呆毛弯成一个爱心形状。配文泡泡从她身边浮出："你下载了吗？" 画面切回现实——手机屏幕跳出五星好评弹窗。拇指在五星全部点满。屏幕上的企鹅露出满意的微笑——但她的眼神方向微微偏离镜头像在看着屏幕外——嘴角带着一丝"我就知道你会下载"的狡黠微笑。画面最终定格在手机屏幕上——她的wink表情被设置成了聊天背景。

梦幻泡泡世界 → 现实手机屏幕的反转，画风突变强调反差冲击 --ar 9:16 --motion 2 --stylize 200
```

## 使用技巧

1. **前段铺垫**足够自然真实让观众相信剧情走向
2. **反转点**在最后几秒突然爆出，前面不暗示
3. **表情弧线**是反转关键：正常→困惑→震惊→认命
4. **分段生成**：两段分别生成后用剪辑拼接更可控
5. **字幕卡点**：后半反转处加配音字幕大幅强化笑点
