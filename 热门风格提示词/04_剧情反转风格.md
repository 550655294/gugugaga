## 提示词 1：咕咕嘎嘎减肥记（喜剧反转·30秒）

```
杰作，3D萌系动画。30秒竖屏。

【前15秒 雄心壮志减肥】
厨房暖黄灯光，企鹅妹妹叉腰站老式指针体重秤前——秤盘有灰说明很久没用。深呼吸闭眼，先右脚踩上指针微动，左脚跟上指针疯转两圈半停住。主观视角：数字被超厚马赛克完全遮蔽，旁贴便利贴"看了会伤心（上次看完哭了三天）"。掏出巨大手绘"减肥计划表"——表格画满红叉和哭泣企鹅emoji，标题"第46次减肥"旁画小鱼干加禁止符号。郑重贴冰箱门用短绳绑死结。转身握紧双翅："gugugaga！"眼中冒两团小火焰。蒙太奇：仰卧起坐起一倒三气喘如牛→面对绿油沙拉发呆叉子戳每片菜叶但不吃→跑步机扶着扶手被传送带拖脚离地水平滑行→跳健身操左右脚完全不在拍子上像跳另一种舞。

【后15秒 深夜偷吃翻车】
时钟快进到凌晨十二点——秒针到12时画面全黑。漆黑中冰箱门悄悄开条缝，内部暖黄灯光如天堂圣光透出照亮她半张脸——眼睛发绿光嘴角邪恶微笑呆毛弯成恶魔角。左右飞速探头确认安全踮脚尖溜到冰箱前。用喙咬断死结，冰箱门大开：成排小鱼干、三盒布丁、奶油蛋糕、大桶冰淇淋、两罐巧克力酱——暖黄灯光把食物照得无比诱人有柔和辉光。暴风吸入——左手鱼干右手蛋糕同时塞嘴两腮鼓如花栗鼠，嘴周沾满奶油饼干渣。突然"啪"一声审讯室惨白顶灯全亮——她僵住嘴里还塞着食物。镜头摇到她身后——墙上密密麻麻贴满微型摄像头，十几个从不同角度对着冰箱。屏幕浮现白色小字："摄像头37号·第47天·凌晨00:03:42"。她缓缓转身面对摄像头阵——满嘴食物含糊说："g...guga？（能删掉吗）"小鱼干尾巴从嘴角探出晃晃。画面定格窘迫全脸——眼睛旋转尴尬圆圈，淡出。

```

## 提示词 2：咕咕嘎嘎的生日（温馨反转·28秒）

```
杰作，3D萌系动画。28秒竖屏。

【前14秒 孤独生日】
傍晚昏暗客厅——窗帘全拉仅一丝缝透橘色路灯。企鹅妹妹独自坐茶几前，巴掌大蛋糕歪歪扭扭插一根数字蜡烛"1"。环顾四周——沙发空荡/饭桌空荡/玄关空荡/无气球彩带无装饰。低头——纸质生日王冠"生日快乐"小旗垂下，王冠边缘沾汗渍。拿起手机——联系人列表全灰0条未读，屏幕暗了点亮暗了再亮第三次没再点。把手机翻面扣桌上——背面贴纸"今天是我的生日！"无人看见。摸出小镜子对镜给自己戴寿星帽，又叠一个成两层王冠。划火柴三次才着，烛火在翠绿瞳孔中摇曳映出火苗。对着烛火小声："gugu..."——声音在空荡客厅形成微小回音。

【后14秒 惊喜变惊吓】
闭上眼深吸气准备吹蜡烛——腮帮鼓到极限——门"砰"撞开。一群企鹅蜂拥而入：领头举比她大三倍彩虹气球束顶着天花板→第二只端七层奢华蛋糕塔奶油裱花极尽华丽顶站糖艺企鹅→第三只头顶几百条小鱼干拼巨幅"生日快乐"字→第四只拿彩带喷射器对准狂轰滥炸。她表情——惊恐瞪眼→眼眶泛红→嘴唇颤抖——所有人以为感动哭——结果她喊"蛋——糕——！！" 慢动作：冲最前面胖企鹅因地滑刹不住脚，连人带脚踩进蛋糕塔——七层蛋糕如多米诺骨牌从下往上倒塌，奶油空中螺旋，整个塔扣她头上。画面巨响静止——她被奶油完全覆盖变白色雪人，只两只翠绿大眼在厚奶油中眨巴睫毛凝奶油霜。企鹅们集体石化——头上鱼干"生日快乐"歪一字变"生日不快"——角落举气球企鹅松手气球飞走默默退出轻轻带门。她伸舌舔嘴周奶油——吧唧嘴："guga（甜的）。"一根蜡烛从奶油中重新燃起照亮裹满奶油的脸。

## 提示词 3：咕咕嘎嘎拯救世界（英雄反转·28秒）

```
杰作，3D萌系CGI动画。28秒竖屏。

【前14秒 末日降临】
黄昏城市全景——高楼大厦平静日常。突然尖锐防空警报响彻全城。高架桥汽车急刹司机探头看天空。镜头沿路人视线缓缓上摇——突破云层巨大陨石破空而下，拖数公里橙红火焰尾迹，表面燃烧岩浆裂纹发地狱暗红。人群四散奔逃，街头屏幕全切换红色紧急警报。新闻直升机空中颤抖，画外记者声："距撞击还有三十秒……二九……二八……"航拍陨石阴影覆盖半个城市，地面震动细纹。

切企鹅妹妹公寓——茶几上吃半包薯片和遥控器。陷沙发里电视直播末日画面，嘴里咬小鱼干咀嚼节奏毫无变化与末日形成诡异平静反差。慢条斯理嚼完舔干净每个翅膀尖用五秒，从沙发滑下来（腿太短脚够不到地像滑下来）。走到鞋柜找出舍不得穿的闪光运动鞋——认真系蝴蝶结，打死结解半天。

【后14秒 史上最潦草拯救】
走到阳台——夕阳陨石火光双重打脸半边暖半边红诡异光效。四处看——从晾衣架取下普通彩色塑料晾衣夹（黄色旧夹/表面褪色纹路，之前夹被单）。把夹子抛两下试手感，深吸气摆超级英雄起手式——一手后拉一手指天/呆毛竖起/运动鞋闪光灯突然亮起。对天空把晾衣夹投掷出去——夹子不可思议速度力量冲向天空突破音障产生小型白色锥形云，飞行轨迹笔直金线。

远景：巨大燃烧陨石占画面90% vs 米粒大黄色塑料夹子几乎看不见。陨石表面特写——竟长出一个鼻子状凸起有清晰鼻孔。夹子精准咔嚓夹在鼻子上——陨石发出惊天"嗷————呜！！！"然后在空中硬拐90度直角擦最高摩天楼天线侧面飞过，消失地平线尽头云层里。留弯折烟痕和空气飘散夹子塑料焦味。

她拍拍翅膀灰从阳台走回房间，捡遥控器关电视。看黑屏中自己头发乱但神情淡然。打开薯片袋发现吃完捏扁放茶几，去厨房重拿一包。最后画面——躺沙发咬小鱼干，电视回放夹飞陨石新闻。主持人激动喊"这是我们唯一画面！"传回视频只有模糊光团——"怀疑外星科技干预"。她面无表情嚼着举起遥控换台变烘焙节目。画面定格草莓蛋糕——她咽口水。

## 提示词 4：咕咕嘎嘎表情包（次元壁破碎反转·26秒）

```
杰作，超可爱3D萌系动画。26秒竖屏。

【前13秒 奇幻世界铺垫】
粉紫色调异次元泡泡空间——天渐变粉地是半透明蓝镜面反射天空。到处飘浮大大小小半透明泡泡——每个泡泡里活着一幅她的小表情：开心眼睛弯月牙、超级生气圆鼓脸蒸包冒出蒸汽、困惑问号头顶旋转一变三、打哈欠张嘴能看到喉咙小舌头、花痴眼中两个巨大心形瞳旋转。她轻盈跳跃——每踩地面泛涟漪飘出音符。跳起戳破"生气脸"泡泡——啪散成七彩光点和音符，表情缩成发光小图标飘进喙里。戳一个又一个——喙边缘发出霓虹光。收集完毕站镜面上低头看——水中倒影和真人隔空互动：她wink→倒影飞吻/做鬼脸→倒影扮酷/两人同时比心→爱心从水面浮出变粉红实体桃心悬浮被她接住抱着玩。

【后13秒 次元壁打破】
突然画面剧烈震动空间扭曲。梦幻世界色彩从渐变被拉成线条——所有泡泡同时破裂化为流水——世界像被卷进抽水马桶缩小缩小再缩小。最后从缤纷泡泡世界缩小成普通智能手机屏幕。一只人类手拿手机拇指滑动——屏幕上微信表情商店页面显示"咕咕嘎嘎表情包第3弹——售价1元"。拇指点击"下载"。画面切回泡泡世界——她突然被无形力量支配身体不受控做出各种表情一个接一个像拍大头贴。试图反抗："我又不是表情包！我要自由！！"但身体强制摆剪刀手wink/嘟嘴撒娇/双手捧脸期待。最后认命——对虚空镜头摆出最完美wink眼带动漫闪亮星星脸晕恰到好处呆毛弯成心形。配文泡泡浮出："你下载了吗？" 切回现实——手机屏幕跳出五星好评弹窗拇指点满五星。屏幕上企鹅露出满意微笑——眼神微偏像看屏幕外嘴角带一丝狡黠："我就知道你会下载"。画面定格手机屏幕——wink表情被设成聊天背景。

梦幻泡泡世界→现实手机屏幕反转，画风突变强调反差冲击 --ar 9:16 --motion 2 --stylize 200
```

## 使用技巧

1. **前段铺垫**足够自然真实让观众相信剧情走向
2. **反转点**在最后几秒突然爆出，前面不暗示
3. **表情弧线**是反转关键：正常→困惑→震惊→认命
4. **分段生成**：两段分别生成后用剪辑拼接更可控
5. **字幕卡点**：后半反转处加配音字幕大幅强化笑点
