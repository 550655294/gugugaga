# 双段对比风格提示词

> 风格来源：抖音"前后对比" / B站"理想vs现实"类创作者
> 核心：AB双段对比、互动对话、反差萌

---

## 风格特征

- 🔄 结构：A段（7秒）+ 转场 + B段（7秒）+ 合体ending
- 🎭 对比维度：理想vs现实 / 白天vs夜晚 / 独处vs社交 / 认真vs摆烂
- 🗣️ 台词：简短互动对话，gugugaga句式贯穿
- 📐 画面：两段色调、打光、配乐形成鲜明对比

---

## 提示词 1：咕咕嘎嘎理想vs现实（上班版）

```
杰作，3D萌系分屏动画，左右双画面同步对比。

【左半屏 理想版 / 暖色调金色滤镜】
企鹅妹妹早晨被阳光轻轻唤醒，优雅地伸个懒腰。她从容地喝了一口咖啡，穿上整洁的小西装，对着镜子自信微笑。然后迈着轻快的步子走出门，阳光洒在她身上形成美丽的光晕。字幕："gugu~ 又是美好的一天"

【右半屏 现实版 / 冷蓝色调手忙脚乱】
同一时刻：闹钟已经响了第八次，她像一滩烂泥趴在床上，一只翅膀无力地拍闹钟却拍到了水杯——水杯倒了，水洒到插线板上（冒出小火花特效）。她弹射坐起来头发像鸟窝，嘴里叼着一片面包就往外冲，结果撞上了门框弹回来。镜头给出玄关处——她穿了两只不同颜色的袜子。

【合体ending 全屏】
地铁站台——左右两边的她同时到达，中间用一道闪电特效分开。理想版优雅等待，现实版趴在地上喘气。闪电消失，两道身影合二为一——她同时戴着领带和顶着鸡窝头，微笑和气喘同时存在，画面产生诡异的叠加故障效果。

最后她对着镜头摊手："gugugaga（懂的都懂）"

```

## 提示词 2：咕咕嘎嘎独处vs社交（社恐版）

```
杰作，3D萌系动画，分屏对比。

【左半屏 独处模式 / 暖黄私人空间】
企鹅妹妹在家里放飞自我：穿着不成对的袜子、把薯片渣吃得满地都是、用翅膀抠鼻孔（被自己发现后心虚收手）、以奇怪的姿势瘫在沙发上刷手机——头和身体扭曲成人类不可能的角度。她看到好笑的视频笑得打滚，从沙发滚到地板上，又滚回来，毫无形象。"gugugaguggaga！"

【右半屏 社交模式 / 粉红社交尴尬】
同一时刻切换到社交场景：闺蜜企鹅来访，门铃响了。右半屏全是0.5秒快速切换——她疯狂收拾房间把垃圾塞进沙发底下、整理发型变换了八种没一个满意、开门时瞬间切换成淑女站姿但额头上还挂着汗珠。她对朋友微笑但眼角在抽搐。朋友说话时她每隔几秒偷瞄沙发上会不会有薯片掉出来。

【合体ending 全屏】
朋友走后，门刚关上她就滑坐在地上长舒一口气，然后一秒恢复放飞自我的姿势。门外传来朋友的声音："下次还来玩哦~"她惊恐瞪大眼睛。

室内暖光，分屏中间有柔和的分割线 --ar 9:16 --motion 3 --stylize 200
```

## 提示词 3：咕咕嘎嘎大人vs小孩（童年版）

```
杰作，3D萌系动画。

【上半段 大人版 / 沉稳色调】
一只成熟的企鹅女士穿着干练的OL套装，夹着公文包走在写字楼里。她处理文件、打电话、喝咖啡的动作优雅而从容。下属企鹅排队报告工作，她点头示意一气呵成。整个画面是高级灰蓝色调，构图稳定。

【转场特效】
```

```
闹钟响第一声，企鹅妹妹就一个弹射起步从被窝里飞出来，精准落地。她神采奕奕，翅膀飞速运作——5秒完成洗漱、3秒穿好衣服、2秒吃完早餐，全程动作如动漫加速播放。出门前对着镜子比了个超元气的V字手势，翅膀尖还带闪光特效。字幕："gugugaga！冲！"

【下半屏 晚安 / 夜晚深蓝紫色调】
同一只企鹅妹妹从门外累瘫了滚进来，像一滩液体一样流进玄关。她四分之三的身体已经进了房间，还有一条腿卡在门外，然后用最后一点力气把那条腿也拖进来，用脚带上门。她趴在地上——不起来了。就用这个姿势像虫子一样蠕动到冰箱旁边，用嘴（喙）拉开冰箱门从最下层叼出一瓶牛奶，喝完后就地倒头睡在冰箱旁边。
```

## 使用技巧

1. **分屏生成**：部分AI工具不支持分屏，建议左右两段分别生成后使用剪辑软件合成
2. **对比要素**：确保AB两段的色调、灯光、情绪形成鲜明反差
3. **转场设计**：用一个画面元素（时钟、日历、门、镜子）作为AB段的连接点
4. **表情弧线**：A段表情 → 转场 → B段表情 → 合体矛盾表情
5. **共鸣感**：双段对比最容易引发"这不就是我吗"的弹幕共鸣
