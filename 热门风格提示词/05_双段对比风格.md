## 提示词 1：咕咕嘎嘎理想vs现实（上班版·28秒）

```
杰作，3D萌系分屏动画，左右双画面同步时间轴对比。28秒竖屏。

【左半屏 理想版 / 暖金色晨光滤镜】
清晨六点半——企鹅妹妹被轻柔的古典乐闹钟唤醒。她在被窝里优雅伸懒腰——被子滑落露出整齐的睡衣（睡衣纽扣全部扣对/没有皱褶），缓缓坐起，下床前还叠好被角。走到全身镜前——镜子里的她精神焕发：头发柔顺光泽每一根都在该在的位置，呆毛翘起完美弧度，肤色白里透粉。刷牙时泡沫都是整齐的圆球，喝咖啡时小指尖微微翘起，三明治切面整齐。穿上熨烫平整的小西装——领带打温莎结一丝不苟，皮鞋能反光。对镜自信微笑——牙齿闪过一道动画闪光。出门时阳光恰好洒在她身上形成一圈金色光晕轮廓——门在她身后自动缓缓关闭。清晨街道上鸟语花香——她轻快蹦跳着过斑马线，每一步落下都有小花从地面冒出。地铁到站时门正好在她面前打开——里面居然有空座。坐下后从公文包拿出书——封面是《高效能企鹅的七个习惯》。

【右半屏 现实版 / 冷蓝灰色调】
同一时刻——闹钟响了第八次（每次按掉又响一次次递进）。她像史莱姆一样趴在床上呈一滩不规则形状——头发炸成鸟窝有十几缕不同方向翘起，嘴边还有干掉的口水痕迹。一只手（翅膀）半死不活地摸向闹钟——摸偏了打翻床头水杯——水灌进插线板冒出一缕小火花和焦焦的烟。她"啾"一声弹射坐起来——被子缠在脚上扯着她先坐再倒回去第三次才挣脱。嘴里叼一块干面包（连包装袋都没拆）冲出卧室——飞到玄关时脚撞到门槛整个人平飞出去在空中转了半圈——鼻子先着地滑了半米远正好滑到鞋柜旁。看一眼钟——瞪大眼睛以快进速度穿袜子（一只蓝的一只红的完全没注意）→ 蹬鞋子（左右穿反了）→ 冲出大门 → 不到一秒又冲回来拿忘带的手机。地铁站——挤到变形——脸贴在玻璃门上被挤成了扁平的企鹅饼，玻璃上印下脸部轮廓雾气。出站后狂奔——跑过同一条斑马线时一只鸟飞过精准地把便便落在她刚洗的衬衫上。

【合体ending 全屏】
地铁站台——左右两边时间线汇合到同一物理空间。理想版优雅站在月台，现实版从电梯方向气喘吁吁手脚并用爬上来。画面中一道闪电特效从中间劈下把屏幕平分成两个世界——金黄vs灰蓝。闪电消失——两个世界的她合二为一：她的外观是理想版（西装整洁）但表情和姿态是现实版（喘不过气/鸡窝头痕迹隐约可见/领带上有咖啡渍不知什么时候沾上的——两个版本的物理特征叠加成一个尴尬的矛盾体）。她面对镜头——想露出理想版的职业微笑但嘴角在抽搐，然后放弃挣扎深呼吸后对着观众摊开双翅："gugugaga（懂的都懂）。"一朵先前的金色小花从理想版世界飘来落在她头上——枯萎变成灰色。
```

## 提示词 2：咕咕嘎嘎独处vs社交（社恐版·26秒）

```
杰作，3D萌系动画，平行分屏对比。26秒竖屏。

【左半屏 独处模式 / 暖黄灯私人天堂】
企鹅妹妹独自在家——房门反锁窗帘拉紧安全感MAX。此刻的她：穿着两只完全不配对的袜子（左脚红白条纹/右脚荧光绿青蛙图案），薯片袋插在肚腩和沙发扶手之间方便随时取用——薯片渣从嘴角掉在肚皮上然后她用翅膀尖粘起来吃回去。姿势——以一个不符合生物力学的奇怪角度瘫在沙发上：头朝下枕着扶手双脚翘在沙发背上，身体形成一个倒U形。用翅膀尖抠了抠鼻孔——被自己发现——心虚地环顾四周确认没人然后快速把手放下假装什么都没发生。刷手机：看到搞笑视频从沙发上笑得滚到地板上 → 在地板上继续笑翻滚了两圈踢翻了垃圾桶 → 又从地板滚回沙发上（动作连贯如一组完美的体操地面动作）。"gugugaguggaga！！"笑声在空中回荡没人管她。

【右半屏 社交模式 / 粉色社交尴尬】
门铃响了。右半屏切换到快速连续剪辑——0.5秒一个镜头：她把薯片袋塞进沙发缝隙→飞到镜子前用手蘸水把翘起的头发压平但只压下去一根另一根翘更高→试了八种发型每一种都比上一种更夸张（从鸡窝→爆炸头→中分→三七分→大背头→丸子头）→放弃→冲去开门前在门口急刹车——从弯腰喘气的狼狈状态瞬间挺直腰板切换成端庄淑女站姿（但额头上还挂着三颗巨大的汗珠）。闺蜜企鹅进门——她对朋友露出完美微笑："gugu~好久不见~"但眼角在不自然抽搐。朋友坐在沙发上——她每隔三秒偷瞄一下薯片是不是从沙发缝里掉出来（没掉→松一口气/有个角露出来→额头又开始冒汗）。茶几上的布丁缺了一个角——她用身体挡住朋友视线并慢慢挪动让身体始终挡在布丁和朋友的连线之间。朋友说去洗手间——她一路护送并指着与洗手间方向相反的路试图误导。

【合体ending 全屏】
朋友终于走了——门关上的声音响起她整个人"哗啦"从紧绷状态散架：肩膀垮掉→脊椎弯成虾→站姿回到独处模式的歪歪扭扭→长长舒了一口气（气量之大把刘海都吹起来了）。正准备瘫回沙发——门外传来朋友的声音："对了！下周还来你家玩哦！我带了新的桌游！"她瞬间石化——全身从脚开始变为灰色石像一直蔓延到脖子。眼睛惶恐地瞪大，嘴唇在微抖。石像状态定格：“嘎……？”

暖黄单人安全区 vs 粉色社交焦虑区，双色切分中间有柔和渐变线 --ar 9:16 --motion 3 --stylize 200
```

## 提示词 3：咕咕嘎嘎大人vs小孩（童年版·24秒）

```
杰作，3D萌系动画，上下分段叙事。24秒竖屏。

【上半段 大人版 / 冷灰蓝沉稳色调】
写字楼高层——落地窗外是城市天际线云层在脚下飘过。一只成熟的企鹅女士——穿着剪裁完美的深蓝色OL套装，领口别一枚银色小鱼胸针，脚踩黑色小高跟走在大理石走廊上。每步节奏精准——鞋跟敲击大理石的声音清脆规律。她夹着黑色公文包走进会议室——推门的力度恰到好处。坐在皮椅上，面前是一排待签文件。她翻阅速度极快，签字笔在纸面游走的动作流畅无一丝多余。接电话——声音沉稳专业："数据报告周五前完成。——嗯。——好。——guga。"挂断。端起咖啡抿一口——放下时杯底与碟子接触没有一丝声响。窗外正午阳光穿过百叶窗在桌面投下平行的条形阴影。

【转场】
她合上公文包起身——包没拿稳从手中滑落。镜头追着公文包向下坠落——包重重砸在地板上发出闷响。镜头在包上停留一秒，然后缓缓上摇——摇到桌腿的高度停住。

【下半段 小孩版 / 明亮蜡笔彩色调】
同一只企鹅变成了迷你版——穿着完全同款但拖地的深蓝色OL套装（袖子卷了六层/裤腿拖在地上变成了扫地的拖把），迷你公文包和她身体一样大她得两只翅膀环抱才能搬动。她在铺满蜡笔和涂鸦纸的地板上学大人走路——模仿高跟鞋的节奏但每走三步就被自己的裤腿绊倒一次。拿起一个玩具电话贴在耳边——发现是根弯弯的香蕉——愣了一秒然后继续对着香蕉严肃地说："数据...嗯...周五。"她面前排了一排企鹅布娃娃——她戴着从玩具医药箱里翻出来的玩具眼镜（没有镜片），用小木棍当教鞭指着白板上的"工作报告"——上面全是歪歪扭扭的简笔画企鹅和鱼。一只布娃娃倒了——她马上停止"会议"小跑过去扶起来拍它的背："不哭不哭。"重新在"白板"上签字——认真地画了一排歪歪扭扭的小鱼干，每一条大小形状都不一样，其中一条只有一只眼睛。此时一只真正的大企鹅（温柔的年长女性）在门口探出头——脸上满是宠爱——然后轻轻敲了敲门框用幼稚园老师的语气："总——经理？我可以进来吗？"迷你企鹅立刻爬上成人尺寸的办公椅（爬了三下才上去），端正坐好——手放膝盖上/背挺直/下巴微抬/呆毛翘起——清了清嗓子用最正式的语气："gugu！请进！"
```

## 提示词 4：咕咕嘎嘎科技vs传统（穿越版·26秒）

```
杰作，3D萌系动画，上下双世界。26秒竖屏。

【上半段 未来科技版 / 赛博霓虹冷色调】
2084年——巨型摩天楼构成的未来都市，高空的悬浮车道光带穿梭在建筑之间，全息投影广告在空中立体漂浮。企鹅妹妹——穿着一件带有发光路径线条的银色紧身衣——站在磁悬浮滑板上从三百楼滑下——滑板尾端喷出蓝色离子焰。她的智能镜片上滚动着信息流：天气/日程/健康数据。回到家——公寓门识别她的虹膜自动打开洒下柔和的欢迎光带。机器人管家从墙内滑出，用机械臂端上一杯分子合成饮品。她走向传送式食物合成机——将一条真空包装的银色小鱼干投进去——机器发出嗡嗡共振声并亮起蓝色光环扫描鱼干。三秒后机器叮一声——输出端滑出一个洁白陶瓷盘：上面放着一块被重新分子排列成枫叶形状的精致鱼冻料理——周围点缀着可食用发光花朵和啫喱小球。她坐在窗边——窗外是无尽的霓虹城市——用刀叉优雅地切下一小块送到嘴里——闭上眼咀嚼。但她的睫毛微微颤动像在品尝却并不完全满意——眉头有极细微的皱褶。

【转场】
机器突然发出刺耳故障音——蓝色光环转红色——冒出一小股黑烟。她面前的料理和数据全部闪烁然后画面出现故障条纹。

【下半段 古代传统版 / 水墨暖黄宣纸色调】
她被传送到了古代江南水乡——青石板路/白墙黛瓦/柳树拂水/远处有乌篷船摇过。她赤脚踩在石板上——石板被太阳晒得微烫——她"嘶"地蹦起来跳到旁边草地上。身上的银色紧身衣变成了粗麻布小衫（但保留了银色领口的科技残余）。她走到集市——木推车上摆满鲜鱼。她掏了掏口袋——没有钱只有三个发光的贝壳（从未来带过来的）。她把贝壳放在老渔夫面前——老渔夫拿起一枚眯眼看了半天，然后牙齿咬了一下确认不是银子，用看傻孩子的眼神还给她——附送一条最小的猫鱼（"拿去吧小家伙"）。她坐在河边的石阶上——脚浸在清凉河水里，手里攥着一根自制竹钓竿。盯着水面浮标——浮标动了三次她都提晚了——第四次她全神贯注一有动静立刻猛提——因为用力过猛整个人向后翻倒顺着石阶滑进了河里。"扑通"——水花溅得高高。被捞上来后——浑身湿透头发贴在脸上坐河岸边，背后老渔夫给她生了一小堆篝火。她抱着膝盖坐在火旁——湿透的衣服被火烘得冒出白色水蒸汽，毛绒身体在热气中逐渐蓬松——从瘦小湿身膨胀成一颗炸毛的毛球像一只愤怒的河豚。老渔夫递给她一条串在树枝上烤得滋滋冒油的河鱼——鱼皮焦黄酥脆。她犹豫了一下接过——小心翼翼地咬了一小口。然后——所有动作同时停止。翠绿色大眼睛瞬间亮起四层高光——瞳孔中炸开无数的星星。她看着手里的烤鱼呆滞两秒——然后大口大口撕咬，嘴角沾着鱼油和焦黑炭屑。对着老渔夫双手竖起两个翅膀尖的大拇指："guga！！" 老渔夫被逗乐抚须大笑，把草帽扣在她湿漉漉的头上——草帽太大遮住半张脸只有喙从帽檐下露出还在不停地嚼。画面定格——篝火映照下的企鹅毛球+草帽+烤鱼+满天繁星。
```

## 提示词 5：咕咕嘎嘎早安vs晚安（双态版·24秒）

```
杰作，3D萌系动画，上下分屏同角色。24秒竖屏。

【上半屏 早安模式 / 清晨蓝金活力色调】
闹钟第一声——企鹅妹妹从被窝中弹射起步（真的弹射——像发射导弹一样整个人从被子中垂直弹起来在空中翻了一圈然后双脚稳稳落地，落地时摆出一个超级英雄的三点着陆姿势）。全程加速播放：刷牙——牙刷在嘴里高速震动冒出白色泡沫云，吐水精确命中排水口→洗脸——一捧水泼脸然后顺带把呆毛也搓了搓呆毛立刻笔直竖起→穿衣服——衣服在空中自动展开她只需跳进去（但实际上是她跳了三次才套中）→早餐——一杯牛奶仰头一饮而尽一滴不洒→整理仪容——对镜子迅速检查牙齿（闪过一道闪光）/呆毛调整（弹了弹）、双手用力拍了拍脸颊把肉拍得晃了晃。出门前——对镜子里的自己比了一个超元气的V字手势——食指和中指分开的瞬间翅膀尖冒出闪光特效——同时镜面炸开一圈放射状光芒背景变成日出大海。她喊出："gugugaga！冲！"每个音节都变成金色实体文字从嘴里弹出来一个比一个大——"gugu"撞上镜框边缘弹回来→"gaga"直接冲破镜框飞出去。然后她以旋风速度冲出家门——身后留下一串灰尘形成的企鹅轮廓。

【下半屏 晚安模式 / 深夜深蓝紫倦怠色调】
凌晨一点——门锁转动声缓慢而沉重。门开了——企鹅妹妹没有走进来而是"流"了进来——她的身体像一滩粘稠的液体从门缝底部流淌入玄关然后缓慢汇聚成形。四分之三的身体已经进了房间——还有一条腿和一只翅膀还卡在门外——挣扎几秒后那条腿主动脱了鞋（没用手自己踢掉的）然后缩进来用脚勾住门边把门缓缓带上。恢复实体形态——她面朝下趴在玄关地板上就不动了。就这样——在地板上像虫子一样——用下巴和膝盖的力量匍匐前进蠕动蠕动蠕动——蠕过玄关→蠕过走廊→蠕到厨房冰箱前。连眼睛都没睁开——凭记忆用喙拉开冰箱门（冰箱内冷光溢出照在她脸上形成蓝白对比），从底层叼出一盒牛奶——不用杯子直接用吸管躺着喝。喝完奶盒随手一放——就地翻身在冰箱旁的地板上缩成一个球睡着了。冰箱冷光还开着——在她身上投下一片蓝色的矩形光。

【合体分割线创意】
上下屏之间的分割线是一条时钟的指针——从早8点开始缓慢转动，转到晚11点停止。指针材质是半透明发光体——上半指向朝阳方向（金黄光）→下半指向深夜方向（蓝紫光）。最终画面上半是她的早安V字姿态定格（元气光芒万丈背景）——下半是她晚安冰箱旁蜷缩睡姿（冷光静谧），中间只隔一根逐渐减慢的指针。早安版和晚安版的她——完全一样的脸完全相反的灵魂。
```

## 使用技巧

1. **分屏生成**：部分AI工具不支持分屏，建议左右两段分别生成后用剪辑软件合成
2. **对比要素**：确保AB两段色调/灯光/情绪形成鲜明反差
3. **转场设计**：用一个画面元素（时钟/日历/门/镜子）作为AB段连接点
4. **表情弧线**：A段表情→转场→B段表情→合体矛盾表情
5. **共鸣感**：双段对比最容易引发"这不就是我吗"弹幕共鸣
