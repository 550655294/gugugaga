# Seedance 2.0 标准格式提示词

> 风格来源：@Sharon Riley / Seedance 2.0 官方案例库
> 公式：画质前缀 + 主体 + 动作 + 场景 + 光影 + 镜头 + 风格 + 画质 + 技术参数

---

## 标准公式

```
杰作，最佳品质，[风格/类型定义]
↓
主体：角色（来源/外貌/服装/体态）—— 分部位细致描述
↓
动作：具体表演（肢体/表情/台词/声音）—— 按时间顺序或事件拆分
↓
场景：背景环境 + 时间/天气 + 光影效果 + 氛围
↓
特效：物理反馈 + 粒子效果 + 材质质感
↓
镜头：景深/角度/运动 + 画幅 + 分辨率 + 技术参数
```

## 提示词 1：咕咕嘎嘎冰上华尔兹（中英双语专业版）

```
杰作，最佳品质，超可爱的3D萌系CGI动画，流畅丝滑的动作。30秒竖屏。

主体：圆滚滚的黑白企鹅妹妹——一身柔软蓬松的黑色毛绒外表，抚摸上去的触感像最细的羊绒，圆润的纯白腹部如一颗饱满的棉花糖。短小可爱的翅膀和黄色蹼足——掌心有粉嫩的肉垫细节。巨大的黄色喙——表面光滑如陶瓷，上喙顶端有一小片微微上翘的弧度。动漫风格的面孔——黑色齐刘海波波头每一缕发丝独立可辨，额上碎发在光下泛着淡蓝光泽。闪闪发光的翠绿色大眼睛——拥有四层高光（主光、辅助光、环形光、底部反射光），瞳孔中的虹膜纹理如万花筒般细腻。长长的睫毛在眨眼时如蝴蝶翅膀般缓慢扇动。粉嫩的脸颊带有自然的渐变色——从中心樱粉向外过渡到奶白。穿着一件闪亮的冰蓝色芭蕾舞短裙——裙摆由三层半透明薄纱组成，最外层缀有银色雪花亮片，腰间系银丝蝴蝶结。脚上是一双白色冰刀鞋——刀片在冰面上划过时会留下微小冰晶尾迹。

动作与表演时间线：
【0-5秒开场】灯光暗下，一束追光从正上方打在冰面中央——光斑直径恰好容纳她的身体。她站在光圈中，背对镜头，头微微垂下。背景有冰晶凝结的细微叮咚声。她缓缓抬起双臂——双翅向两侧伸展，指尖绷直，如芭蕾舞者起手式。冰刀在冰面上轻轻一点——一圈涟漪状冰纹从刀尖扩散开来。
【5-12秒第一段动作】她开始滑行——在镜面般的冰面上画出一道完美弧线，冰刀与冰面的摩擦声轻柔悦耳。先是一组优雅的单脚旋转——左脚为轴身体缓缓转动，右脚在后保持平衡。裙子随着旋转展开如一朵盛开的冰蓝色花朵——三层薄纱在旋转中层次分明重叠交错，雪花亮片将追光反射成无数细小的光点投射在冰面和周围空气中。然后是一个笨拙的三周旋转——第一圈身体笔直姿态完美，第二圈手臂开始晃动裙摆出现不规则波纹，第三圈结束时平衡彻底崩坏——她"扑通"一声摔坐在冰面上。坐在地上的瞬间——冰蓝色短裙像降落伞一样在周围蓬开铺成一个完美的圆形，亮片闪烁着无辜的光芒。
【12-18秒起身挣扎】她坐在地上愣了一秒——脸上是"发生了什么"的呆滞表情。然后不好意思地用翅膀捂住整张脸——从羽毛指缝中偷偷露出一只眼睛瞄向镜头。当她尝试起身时——左脚冰刀刚撑住冰面右脚就打滑，双腿开始在冰面上快速原地奔跑——频率快得像卡通片里的加速循环动画，但身体纹丝不动。脚下被她刨出了一道带有冰屑的浅沟——冰屑在她身后堆积成了小雪山。她终于放弃挣扎趴在了冰面上——脸颊贴着冰，鼻孔呼出的白气在冰面上形成两小块雾霜，双手在前面画了一个委屈的小圈圈。
【18-25秒意外优雅】趴在地上时她无意中推了一下冰面——身体竟以肚子为轴开始在冰面上缓缓旋转起来。旋转速度不快，但姿态出人意料地优雅——身体平贴在冰面上四肢自然伸展，短裙平铺像一个冰蓝色的大圆盘。她在旋转中从困惑→惊喜→享受，最后索性闭上眼睛微笑，任由身体在冰面上缓缓转圈。旋转的冰刀在冰面画出渐开的螺旋纹——从中心向外扩散，纹路纤细如发丝。
【25-30秒收尾】旋转缓缓停止，她躺在冰面正中央——胸口起伏喘气但脸上是满足的笑容。镜头从正上方俯拍——她躺在自己创作的螺旋冰纹中心，冰蓝色短裙如花朵般铺开。片片雪花从上方缓缓飘落——有几片落在她的睫毛上、鼻尖上、伸出的翅膀尖上。她对着上方的镜头眨了一下眼——睫毛上的雪花被弹向空中变成细碎冰晶。画面渐渐淡出至白。

场景与光影：冬季的户外冰湖，湖面平滑如天然镜面反射着月光。周围是白雪覆盖的松树林——树枝上挂着冰棱和圣诞彩灯，暖黄色小灯在白色世界中星星点点闪烁。夜空是深邃的靛蓝色——满月高高悬挂投下柔和的银白月光，月光在冰面形成梦幻的蓝白色光晕反射同时照在薄纱裙上产生半透明的透光效果。远方有一座覆雪的小木屋，烟囱飘出袅袅白烟。

细节与特效：她每次呼吸都从喙中吐出可见的白色雾气——雾团在空中成形散开。冰刀划过冰面时有细小的冰晶飞溅——每颗冰晶如微型钻石在空中闪烁。摔倒时溅起的冰屑在月光下形成银色瀑布。月光穿过她半透明芭蕾舞裙时产生柔和的光线衍射——裙摆边缘有淡淡的蓝白色辉光。滑行轨迹上留下一道细细的冰晶尾迹——尾迹中包裹着被刀锋碾碎的微观气泡。她摔倒时冰面上会留下她身体的完整印痕——连裙子亮片凹痕都清晰可见。

电影级景深——前景冰屑保持锐利中景人物清晰背景渐虚。柔和的冬季蓝白色主调，暖黄灯光在画面各处点缀形成冷暖对比。体积光笼罩整个场景——月光穿过松枝松针投下斑驳的光柱。温馨奇幻氛围，极高纹理细节，4K分辨率。

## 提示词 3：咕咕嘎嘎沙滩派对（夏日活力版）

```
杰作，最佳品质，3D萌系动漫动画，流畅生动的动作，夏日活力氛围。30秒竖屏。

主体：企鹅妹妹在阳光灿烂的热带海滩——戴着超大的圆形墨镜（镜片是渐变色从深棕到透明，镜框比她脸大三圈所以不停从鼻梁滑下来，她隔几秒就得用翅膀尖推一下鼻梁位置）。头上一顶米色小草帽——帽檐上插了一朵鲜红色扶桑花，花蕊里还有一颗小水珠。防晒霜抹得满脸白白的——鼻梁上一道没涂匀的白条，额头上有几个白色手指印（因为翅膀不好操作）。脖子上挂着一个充气粉红火烈鸟游泳圈——火烈鸟的脖子弯成"S"形，黑色塑料眼睛歪了一只。肚子上贴了一张企鹅形状的防水创可贴。

动作时间线：
【0-5秒 沙滩埋自己】她趴在蓝色沙滩垫上——垫子四角用四个贝壳压住。手里拿着一把小塑料铲子（铲柄是冰淇淋造型）挖沙子往自己身上盖——动作极其认真，每次铲沙前还要用铲背把沙拍平整。从脚开始往上盖——盖到肚子时停下来在肚子位置堆了一个沙制的迷你城堡，城堡尖顶插了一面小旗帜（从鸡尾酒杯里抽出来的纸伞）。盖到翅膀时手不够长——用嘴叼铲子从侧面往身上扬沙。最后只露出戴墨镜的脸和黄色喙在沙堆外——整个沙堆是完美的企鹅身体轮廓（利用下方真实身体打底上面精准堆沙塑形）。此刻一只派大星粉色的寄居蟹背着螺旋壳从她的沙堆旁爬过。她闭着一只眼从墨镜缝隙里偷瞄蟹——然后悄悄从沙堆里伸出一根翅膀尖——戳了戳蟹壳。寄居蟹二话不说钳子直接夹住她翅膀尖。她钻出沙堆——沙砾满天爆炸——站起来疯狂甩翅膀被蟹夹着凌空飞舞。寄居蟹最终被甩飞——在空中飞速旋转啪嗒一声精准落回它原来的位置继续爬走仿佛什么都没发生。她的翅膀尖留下一个完美的蟹钳形红印。

【5-12秒 入水恐惧症】沙堆炸开后她拍拍身上的沙——沙粒在阳光下闪着金光从她毛绒身体上簌簌掉落。站起来戴着火烈鸟游泳圈奔向大海——小碎步跑起来圆滚滚身体左右摇摆幅度极大。跑到水边——第一波浪花只有她脚踝高度但打过来时她尖叫着后跳一米远——双手高举踮脚尖后退。又试了一次——这次用一根脚趾试探水面温度，脚趾刚碰到水就缩回来在另一条腿上蹭了蹭。再试第三次——闭紧双眼深吸一口气（吸得火烈鸟游泳圈被撑大变紧）大喊"gugugaga！！"然后冲进水里——水只到她肚子高度。她站水中静默两秒——缓缓睁眼，低头看水面刚好在大白肚子下方——表情从恐惧→困惑→如释重负→得意。"嘎！"叉腰挺肚站在水里摆出征服大海的姿势，一个更小的浪花从侧面偷袭她——水花拍在她脸上，墨镜被冲歪挂在喙上。

【12-20秒 沙滩排球灾难】画面切到沙滩排球网——网是儿童尺寸的高度刚好到她头顶。她戴着火烈鸟游泳圈站在网左侧摆出专业排球姿势——压低重心翅膀前伸，但火烈鸟的脖子刚好挡住她视线。对面飞来的球是轻飘飘的彩色沙滩球——她深吸一口气猛然起跳——跳离地面三厘米（弹跳力被游泳圈限制），头刚好过网，用头正面接球——球弹飞，她也被反作用力弹得向后翻了一圈——游泳圈撞沙滩，她仰面朝天四脚晃在空中如被翻过来的乌龟。挣扎一阵无法翻身——最后手脚并用像不倒翁一样左晃右晃终于弹回正面坐起来——脸上全是沙，墨镜挂在耳朵上，小草帽扣在她脸上只露出喙。

【20-30秒 日落告别】黄昏降临——天空从橙渐变到粉再到淡紫，太阳在海平线上只剩一半。金色余晖铺遍整个沙滩和海面——海面变成一面熔金镜子。她坐在沙滩毛巾上——身上裹着一条白色浴巾（浴巾对她来说像一件拖地长袍），翅膀捧着一颗刚敲开的椰子——椰子上插了两根扭扭吸管。旁边沙地上她用贝壳摆了一排——从大到小像全家福。远处海面上有帆船缓缓驶过——船帆被夕阳染成橘红色。她对着夕阳举起椰子和海平线上的太阳干杯。然后转头看镜头——墨镜里的倒影是这个最完美的夏日黄昏，露出灿烂笑容："guga！"画面定格。一颗排球在一旁被海浪缓缓冲上岸边——滚到她脚边停下。

场景与光影：热带白沙滩沙子细腻如糖粉——沙中夹杂着细碎的天然贝壳颗粒在阳光下闪烁。翡翠色清澈海水——从岸边浅蓝渐变到深处深蓝，能看到水下白沙滩的纹理和小鱼群的影子。远处棕榈树在热风中摇曳——叶影在沙滩上缓缓移动。正午时分强烈阳光在沙子上形成硬朗清晰的高反差阴影。空气中因热浪产生轻微的折射波动效果——远方的景物有微小变形。水面上反射着成百上千的金色光斑——形成一片闪光的星海，光斑随波浪轻轻晃动明灭。

质感与特效：沙粒粘在她毛绒身体上的颗粒感——身体干了后沙粒一颗颗脱落。水花飞溅成透明弧面——每一滴水珠如微型透镜折射着阳光产生七彩。防晒霜的白色膏体不均匀涂抹的质感。火烈鸟游泳圈的充气塑料表面——太阳照射下表面有一层塑料特有的反光。墨镜镜片上的渐变色彩和反射景物。她浑身湿透后毛绒身体缩小一圈——水珠顺着羽毛尖滴落的慢镜头——每滴水珠离开身体时拉出一条细丝。

电影级景深——前景沙地纹理和后景海面虚化衬托人物。鲜艳饱和的热带色彩但保留自然感。阳光直射下的硬阴影+日落时分的金色柔光两段光线对比。体积光穿过棕榈叶投下斑驳的影子。高细节纹理。夏日欢乐温暖氛围。4K分辨率。

## 使用技巧

1. **分层编写**：主体→动作→场景→特效→技术参数，AI能更好理解
2. **动作时间线**：用 `[0-5秒]` 标注帮助AI控制节奏
3. **细节翻倍**：颜色/材质/光影每个维度都展开描述
4. **英文版优势**：Seedance 2.0 英文训练数据更丰富，英中双语效果好
