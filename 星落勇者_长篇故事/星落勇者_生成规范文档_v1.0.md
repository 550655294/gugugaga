## 一、世界观

### 🌍 艾斯特拉大陆（Astra Continent）

千年一遇的**星落之夜**，星辰碎片自天穹坠落，选中凡世生灵成为「星选勇者」。
这一次，星辰没有落在人类王国——它落进了极北的企鹅村落。

**核心冲突：** 深渊之王被封印千年，企图夺取星辰碎片的力量打破封印，吞并整片大陆。

## 五、对话系统（咕嘎 + 叮咚翻译 + 史诗旁白）

### 🔑 核心规则

> 妹妹**只说"咕嘎"变体**，叮咚**翻译成人话**。两者的台词在提示词中成对出现。
> 每集穿插**史诗旁白**（日文或英文），增强世界观的厚重感与仪式感。

### 🎙️ 旁白（Narrator）— 星辰之声

| 属性 | 设定 |
|------|------|
| 身份 | 无名叙述者，如同星空中回荡的古老声音，讲述艾斯特拉大陆的命运 |
| 声线 | 深沉浑厚的男声（后期配音用），如纪录片旁白或动画预告解说 |
| 语言 | **日文**（主，动漫史诗感）或 **英文**（辅，国际化），每集选用一种 |
| 出现时机 | ① 开场白（每集必有）② 关键转折（1-2次/集）③ 片尾结语 |
| 风格 | 短促有力、充满仪式感——像《进击的巨人》旁白或《最终幻想》开场白 |
| 字数 | 每段旁白 ≤ 30 字（中文），≤ 20 词（日/英） |

### 旁白出现时机

| 时机 | 位置 | 示例（日文） | 示例（中文意译） |
|------|------|------|------|
| 📖 开场白 | 场景一开头 | `星は、落ちた。` | 「星辰，坠落了。」 |
| ⚡ 关键转折 | 敌人登场/力量觉醒/绝境反转 | `その小さき翼が、運命を変える。` | 「那双小小的翅膀，将改变命运。」 |
| 🔚 片尾结语 | 收尾段末 | `かくして、勇者たちの旅は始まった。` | 「于是，勇者们的旅途开始了。」 |

### 旁白语言选择规则

> 每集**固定选用一种**，不在同一集中混用日文和英文旁白。

| 集数 | 推荐语言 | 理由 |
|:--:|:--:|------|
| 1 | 日文 | 开篇用日文奠定史诗基调 |
| 2-5 | 英文/日文交替 | 冒险展开，英/日轮换保持新鲜感 |
| 6 | 日文 | 古龙遗迹·传承回，日文仪式感最强 |
| 7-9 | 日文 | 高潮段落，日文张力最足 |
| 10 | 英文 | 终章用英文收束，国际化的告别 |

### 对话格式模板（含旁白）

```
【旁白】日: 『星は、落ちた。』  / 英: "The star has fallen."

妹妹[动作+表情]：「咕嘎！」
叮咚眨动金眼：「[翻译内容]」
```

### 翻译风格

| 妹妹的咕嘎变体 | 情绪 | 叮咚翻译示例 |
|------|------|------|
| 咕嘎！ | 坚定/宣告 | 「放马过来。」「我们上。」 |
| 咕嘎咕嘎！ | 兴奋/赞同 | 「太棒了！」「就是这个！」 |
| 咕……嘎…… | 委屈/失落 | 「好累……」「为什么……」 |
| 咕嘎？ | 疑问 | 「什么意思？」「你是谁？」 |
| 咕嘎~~~ | 满足/幸福 | 「好舒服~」「赢了~」 |
| 咕嘎！咕嘎！ | 着急/慌乱 | 「快跑！」「来不及了！」 |

### 叮咚的独立台词

叮咚不仅翻译，偶尔有自己的吐槽/解说（每集≤3次独立发言，其余是翻译）：

| 场景 | 叮咚独立发言示例 |
|------|------|
| 妹妹莽撞冲进敌群 | 「唉，她又不等我说完冲了。」
| Doro面无表情一击必杀 | 「……有时候真的不知道她在想什么。」
| 菲比施法翻车炸了自己 | 「菲比小姐的魔法事故，本集第三次了。」
| 战斗胜利 | 「冰、炎、光——三人齐了，还有谁不服？」

### 每段必须包含的要素

1. **奇幻场景氛围**：元素光效、魔法粒子、场景特征
2. **具体肢体动作**：战斗/魔法/移动的详细描述
3. **面部表情变化**：眼睛/呆毛/腮红/叮咚眨眼同步
4. **拟声词+翻译**：咕嘎语 + 叮咚翻译成对出现
5. **道具/魔法互动细节**：冰晶之剑、炎尾、光弧等特效
6. **光线/环境细节**：奇幻光影
7. **旁白标注**：开场/转折/结语处标注【旁白】及日文或英文台词

### 密度铁律

| 段落 | 字数要求 |
|------|----------|
| 每段 3s | 180-280 汉字 |
| 收尾段 | 200-300 汉字 |
| 每个场景总字符 | ≤2000 字 |

### 奇幻特殊规则

- 战斗描写保持Q版化——攻击动作是"劈""斩""撞""砸"，不是"刺穿""撕裂""砍断"
- 魔法特效描述要素：颜色 + 形态 + 光晕 + 粒子/余烬
- 怪物保持Q版比例，不可写实恐怖
- 叮咚的翻译/吐槽在3s段中单独占1-2句，不影响动作密度

## 十、即梦生成参数（奇幻版）

```
模型：Seedance 2.0 Fast VIP | 参考模式：全能参考
比例：9:16（竖屏） | 时长：15s × 2段 = 30s/集 | 输入：图生视频(角色参考图+英文提示词)
模式：Agent模式，8段精确控制（每段3s）
积分预估：2段 × 200 = 400积分
提示词长度：35-80 英文单词（英文段）
迭代策略：一次只改镜头/动作/光线其中一个变量
```
